# NetMon 软件部署手册

## 1. 部署模式

NetMon 提供三种部署模式：

| 模式 | 用户数 | 复杂度 | 适用场景 |
|------|---------|--------|----------|
| CLI 模式 | 1 | 低 | 终端直接运行 |
| Web 单用户 | 1 | 中 | 单用户 Web 界面 |
| Web 多用户 | 多 | 中高 | 多用户并发 |

## 3. 模式一：CLI 终端模式

### 3.1 快速启动

```bash
cd /Users/ben/.openclaw/workspace/netmon/core

# 默认扫描 (192.168.1-254.1-254)
python3 realtime_subnet_monitor.py

# 指定参数扫描
python3 realtime_subnet_monitor.py \
  --ip-prefix 192.168 \
  --start-subnet 1 \
  --end-subnet 254 \
  --scan-ip-start 1 \
  --scan-ip-end 64 \
  --mode ping
```

### 3.2 参数说明

| 参数 | 缩写 | 默认值 | 说明 |
|------|------|--------|------|
| --ip-prefix | - | 192.168 | 网络前缀 |
| --start-subnet | - | 1 | 起始子网 |
| --end-subnet | - | 254 | 结束子网 |
| --scan-ip-start | - | 1 | IP 起始 |
| --scan-ip-end | - | 254 | IP 结束 |
| --mode | - | ping | 扫描模式 |
| --tcp-port | - | 22 | TCP 端口 |
| --refresh-interval | - | 120 | 刷新间隔 |
| --subnets-per-group | - | 4 | 每组子网数 |

### 3.3 交互操作

| 按键 | 功能 |
|------|------|
| r | 手动触发扫描 |
| m | 切换扫描模式 |
| q | 退出程序 |
| a-h | 设置项 |

## 4. 模式二：单用户 Web 模式

### 4.1 安装依赖

```bash
cd /Users/ben/.openclaw/workspace/netmon/web
pip3 install flask
```

### 4.2 启动服务

```bash
# 默认端口 80
sudo python3 netmon_web.py

# 指定端口
sudo NETMON_WEB_PORT=8080 python3 netmon_web.py

# 后台运行
nohup sudo python3 netmon_web.py > /var/log/netmon.log 2>&1 &
```

### 4.3 访问

```
http://<服务器IP>/
```

### 4.4 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/status | GET | 获取状态 |
| /api/scan | POST | 开始扫描 |
| /api/scan | DELETE | 停止扫描 |
| /api/results | GET | 获取结果 |
| /api/groups | GET | 获取分组 |
| /api/config | GET/POST | 配置管理 |

### 4.5 API 认证

```bash
# 设置 API 密钥
export NETMON_API_KEY=your-secret-key

# 请求时携带
curl -H "X-API-Key: your-secret-key" http://localhost/api/status
```

## 5. 模式三：多用户 Web 模式

### 5.1 安装依赖

```bash
cd /Users/ben/.openclaw/workspace/netmon/multi/api
pip3 install flask
```

### 5.2 启动服务

```bash
# 默认端口 80，最多 5 并发扫描
sudo python3 netmon_web_multi.py

# 自定义配置
export NETMON_WEB_PORT=8080
export NETMON_API_KEY=your-secret-key
export NETMON_IP_PREFIX=10.0
sudo python3 netmon_web_multi.py
```

### 5.3 多用户支持

- 自动创建用户会话
- 每个用户独立扫描实例
- 通过 `X-User-ID` 或 `user_id` 参数区分

```bash
# 创建新用户扫描
curl -X POST "http://localhost/api/scan?user_id=user-123" \
  -H "X-API-Key: your-secret-key"

# 获取指定用户结果
curl "http://localhost/api/results?user_id=user-123" \
  -H "X-API-Key: your-secret-key"
```

## 6. 生产环境部署

### 6.1 使用 Nginx 反向代理

```nginx
# /etc/nginx/sites-available/netmon
server {
    listen 80;
    server_name netmon.example.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/netmon /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### 6.2 Systemd 服务 (Linux)

```ini
# /etc/systemd/system/netmon.service
[Unit]
Description=NetMon Network Monitor
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/Users/ben/.openclaw/workspace/netmon/web
ExecStart=/usr/bin/python3 /Users/ben/.openclaw/workspace/netmon/web/netmon_web.py
Restart=always

[Install]
WantedBy=multi-user.target
```

启用服务：
```bash
sudo systemctl daemon-reload
sudo systemctl enable netmon
sudo systemctl start netmon
```

### 6.3 Docker 部署 (可选)

```dockerfile
# Dockerfile
FROM python:3.10-slim

WORKDIR /app
COPY . /app

RUN pip install flask

EXPOSE 80
CMD ["python3", "netmon_web.py"]
```

构建运行：
```bash
docker build -t netmon .
docker run -d -p 80:80 --name netmon netmon
```

## 7. 配置示例

```bash

  --ip-prefix 192.168 \
  --start-subnet 1 \
  --end-subnet 254 \
  --scan-ip-start 1 \
  --scan-ip-end 64 \
```

```bash
python3 realtime_subnet_monitor.py \
  --mode tcp \
  --tcp-port 22 \
  --ip-prefix 192.168 \
  --start-subnet 1 \
  --end-subnet 10
```

```bash
sudo apt install nmap

python3 realtime_subnet_monitor.py \
  --mode nmap \
  --ip-prefix 192.168 \
  --start-subnet 1 \
  --end-subnet 254
```

## 8. 安全建议

### 8.1 防火墙

```bash
# 仅允许内网访问
sudo ufw allow from 192.168.0.0/16 to any port 80
sudo ufw allow from 10.0.0.0/8 to any port 80
sudo ufw enable
```

### 8.2 API 密钥

- 生产环境务必设置 `NETMON_API_KEY`
- 定期更换密钥
- 使用 HTTPS 传输

### 8.3 权限最小化

- 避免使用 root 运行 Web 服务
- 创建专用用户 `netmon`
- 限制日志文件权限

## 9. 性能调优

### 9.1 大规模扫描参数

| 场景 | 参数建议 |
|------|----------|
| 20K IP | `--refresh-interval 180` |
| 高并发 | 修改代码 `max_threads = 1000` |
| 低延迟网络 | `--retry-count 1` |

### 9.2 多用户限制

```python
# 在代码中修改并发数
scan_executor = ThreadPoolExecutor(max_workers=10)  # 改为 10
```

## 10. 故障排查

### 10.1 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| ModuleNotFoundError | 缺少 flask | pip3 install flask |
| Permission denied | 端口被占用 | 使用其他端口 |
| ping 超时 | 防火墙阻止 | 检查防火墙规则 |
| No such file | 路径错误 | 检查工作目录 |

### 10.2 日志位置

| 模式 | 日志位置 |
|------|----------|
| CLI | 当前目录 *.log |
| Web | /var/log/netmon.log |
| Docker | docker logs netmon |

### 10.3 调试模式

```bash
# 启用调试输出
export NETMON_DEBUG=true
python3 netmon_web.py
```
