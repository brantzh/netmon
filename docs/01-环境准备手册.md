# NetMon 环境准备手册

## 1. 概述

本文档介绍 NetMon 网络监控软件的运行环境准备要求。

### 1.1 软件架构

| 组件 | 说明 | 位置 |
|------|------|------|
| realtime_subnet_monitor.py | 核心扫描引擎 | core/ |
| netmon_web.py | 单用户 Web API | web/ |
| netmon_web_multi.py | 多用户 Web API | multi/api/ |

---

## 2. 硬件要求

### 2.1 最低配置

| 资源 | 要求 | 备注 |
|------|------|------|
| CPU | 2 核 | 越多扫描越快 |
| 内存 | 2 GB | 20K IP 约需 10 MB |
| 磁盘 | 1 GB | 日志文件 |
| 网络 | 100 Mbps | 扫描速度瓶颈 |

### 2.2 推荐配置

| 资源 | 推荐 | 场景 |
|------|------|------|
| CPU | 4+ 核 | 并发扫描 |
| 内存 | 4 GB | 多用户/大网络 |
| 磁盘 | 5 GB | 长期运行 |
| 网络 | 1 Gbps | 大规模扫描 |

---

## 3. 操作系统

### 3.1 支持的操作系统

| 系统 | 版本 | 状态 |
|------|------|------|
| macOS | 12+ | ✅ 支持 |
| Linux | Ubuntu 20.04+ / Debian 11+ | ✅ 支持 |
| Windows | 10/11 (WSL2 推荐) | ✅ 支持 |

### 3.2 Windows 注意事项

- 推荐使用 WSL2 (Windows Subsystem for Linux)
- 或使用 Git Bash / PowerShell
- ping 命令需系统 PATH 中可用

## 4. 依赖软件

### 4.1 必需依赖

| 软件 | 用途 | 版本要求 |
|------|------|----------|
| Python | 运行环境 | 3.8+ |
| pip | 包管理 | 随 Python 安装 |

### 4.2 可选依赖

| 软件 | 用途 | 说明 |
|------|------|------|
| nmap | 高级扫描 | `sudo apt install nmap` |
| Flask | Web 服务 | 自动安装 |
| Nginx | 反向代理 | 生产环境推荐 |

### 4.3 安装必需依赖

```bash
# macOS
brew install python3

# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# 验证安装
python3 --version
pip3 --version
```

## 5. 网络要求

### 5.1 扫描范围

| 参数 | 默认值 | 说明 |
|------|--------|------|
| IP 前缀 | 192.168 | 网络号.子网号 |
| 子网范围 | 1-254 | 第三段范围 |
| IP 范围 | 1-254 | 第四段范围 |

### 5.2 防火墙要求

| 端口 | 用途 | 说明 |
|------|------|------|
| 22/TCP | SSH | 远程管理 |
| 80/TCP | HTTP | Web 访问 |
| 443/TCP | HTTPS | 安全访问 |
| ICMP | Ping | 扫描协议 |

### 5.3 权限要求

- **Ping 扫描**：无需 root 权限
- **TCP 扫描**：无需 root 权限 (1024+ 端口)
- **nmap 扫描**：建议 root 权限 (ARP 探测)

## 6. 环境变量 (可选)

| 变量 | 默认值 | 说明 |
|------|--------|------|
| NETMON_IP_PREFIX | 192.168 | 网络前缀 |
| NETMON_START_SUBNET | 1 | 起始子网 |
| NETMON_END_SUBNET | 254 | 结束子网 |
| NETMON_SCAN_IP_START | 1 | IP 起始 |
| NETMON_SCAN_IP_END | 254 | IP 结束 |
| NETMON_SCAN_MODE | ping | 扫描模式 |
| NETMON_TCP_PORT | 22 | TCP 端口 |
| NETMON_REFRESH_INTERVAL | 120 | 刷新间隔(秒) |
| NETMON_RETRY_COUNT | 2 | 重试次数 |
| NETMON_API_KEY | - | API 密钥 |
| NETMON_WEB_PORT | 80 | Web 端口 |

## 9. 常见问题

### Q1: macOS 提示 "Permission denied"

**解决**：无需特殊权限，Python 即可运行

### Q2: Windows 下 ping 超时

**解决**：检查 Windows 防火墙设置，允许 ICMP

### Q3: nmap 不可用

**解决**：
```bash
# Ubuntu
sudo apt install nmap

# macOS
brew install nmap
```
