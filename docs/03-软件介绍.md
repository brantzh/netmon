# NetMon 软件介绍

## 1. 概述

**NetMon** 是一款专为高性能计算 (HPC) 和 GPU 集群环境设计的网络监控工具。它能够实时监控大规模子网中的 IP 地址活跃状态，帮助运维团队快速发现和管理集群中的在线设备。

### 1.1 核心特性

- ⚡ **高性能扫描**：支持单次扫描 20,000+ IP 地址
- 🔄 **多模式扫描**：支持 Ping / TCP / Nmap 三种扫描模式
- 👥 **多用户支持**：支持多用户并发扫描，每个用户有独立的扫描实例
- 🌐 **Web 界面**：提供直观的 Web 界面和 REST API
- 🛡️ **线程安全**：经过 v2.0 修复，支持并发安全操作

---

## 2. 应用场景

### 2.1 大规模 HPC/GPU 集群监控

在 HPC/GPU 集群环境中，经常需要监控大量计算节点的在线状态：

| 场景 | 说明 |
|------|------|
| **GPU 集群节点监控** | 监控数百台 GPU 服务器的 IP 在线状态 |
| **HPC 计算节点管理** | 实时监控 1000+ 计算节点的存活状态 |
| **存储服务器监控** | 监控 NFS/GPUFS 存储服务器的在线状态 |
| **网络设备监控** | 监控 IB 交换机、登录节点、监控节点 |

| 集群规模 | 子网数 | IP 范围 | NetMon 适用性 |

## 3. 软件能力

### 3.1 扫描模式

| 模式 | 原理 | 适用场景 |
|------|------|----------|
| **Ping** | ICMP 探测 | 快速发现在线主机 |
| **TCP** | 端口连接探测 | 需要服务发现的场景 |
| **Nmap** | ARP/Nmap 探测 | 局域网最可靠探测 |

### 3.2 部署模式

```
┌─────────────────────────────────────────────┐
│           NetMon 部署模式                    │
├─────────────────────────────────────────────┤
│                                             │

│  ├── 实时显示扫描进度                       │
│  └── 适合快速排查                           │
│                                             │
│  模式二: 单用户 Web                         │
│  ├── Web 界面                              │
│  ├── REST API                              │
│  └── 适合日常监控                          │
│                                             │
│  模式三: 多用户 Web                         │
│  ├── 多用户并发                            │

│  └── 适合团队运维                          │
│                                             │
└─────────────────────────────────────────────┘
```

### 3.3 API 能力

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/scan` | POST | 触发扫描 |
| `/api/scan` | DELETE | 停止扫描 |
| `/api/results` | GET | 获取结果 |
| `/api/groups` | GET | 获取分组 |
| `/api/config` | GET/POST | 配置管理 |
| `/api/status` | GET | 状态查询 |

## 7. 快速开始

### 7.1 CLI 模式

```bash
cd /Users/ben/.openclaw/workspace/netmon/core

# 默认扫描
python3 realtime_subnet_monitor_v2.py

# 扫描 /26 子网 (64 IP)
python3 realtime_subnet_monitor_v2.py \
  --ip-prefix 192.168 \
  --start-subnet 1 \
  --end-subnet 254 \
  --scan-ip-start 1 \
  --scan-ip-end 64
```

### 7.2 Web 模式

```bash
# 单用户版
cd /Users/ben/.openclaw/workspace/netmon/web
sudo python3 netmon_web_v2.py

# 多用户版
cd /Users/ben/.openclaw/workspace/netmon/multi/api
sudo python3 netmon_web_multi_v2.py
```

### 7.3 API 调用

```bash
# 触发扫描
curl -X POST http://localhost/api/scan

# 获取结果
curl http://localhost/api/results

# 获取分组
curl http://localhost/api/groups
```
